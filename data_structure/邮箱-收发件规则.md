### 邮箱-收发件规则-执行条件内容参数结构
[
    {
        "andOr":"and", // 满足以下所有条件：and，满足以下任一条件：or
        "conditionType":1, // 条件运算符 
        "value":"", //值
        "columnName":"sender", //字段名称
        "packetId": 1 //客户分组ID
    }
]


### 条件运算符（conditionType）
1.包含 2.不包含 3.隶属于 4.等于 5.不等于

### 字段名称（columnName）
如果发件人：sender 
如果收件人：recipient 
如果抄送人：cc 
如果主题中：subject 
如果正文中：body
